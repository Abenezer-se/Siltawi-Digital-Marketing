# Siltawi Digital Marketing Website

Official company profile website for Siltawi Digital Marketing. The website showcases the company's services, portfolio, team members, client testimonials, and contact information through a modern and responsive user interface.

## Run Locally

**Prerequisites:** Node.js

### 1. Clone the repository

```bash
git clone <repository-url>
```

### 2. Navigate to the project folder

```bash
cd Siltawi-Digital-Marketing
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Open in your browser

```text
http://localhost:5173
```
